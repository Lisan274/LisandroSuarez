# Productos Bancarios - Angular 15

Aplicacion web desarrollada para prueba tecnica, para la administracion de productos financieros.

Funcionalidades: El proyecto permite listar, buscar, agregar y eliminar productos bancarios.

## Instrucciones de ejecucion
### 1. Ejecutar Backend.
Asegurarse de que el backend este ejecutandose, si no, en el proyecto repo-interview ejecutar:
npm run start:dev

### 2. Instalar dependencias (Backend)
En terminal, en la raiz del proyecto de backend, ejecutar:
npm install

### 3. Instalar dependencias (Angular)
En terminal, en la raiz del proyecto de angular, ejecutar:
npm install

### 4. Ejecutar el proyecto (Angular)
Con angular ya configurado, ejecutar:
npm start

### 5. Ejecutar pruebas unitarias 
Terminar el proceso de angular con Ctrl+c en la terminal o abrir otra terminal y ejecutar:
npm test
